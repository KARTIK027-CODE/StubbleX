# StubbleX

**Turn Waste into Wealth.**

StubbleX is a revolutionary marketplace that connects farmers with industries to turn crop residue (parali) into revenue. 
Instead of burning stubble and causing pollution, farmers can sell it directly to bio-energy plants and paper mills through our platform.

## Features

- **For Farmers**: 
  - AI-based instant valuation of crop waste.
  - Direct listing and selling.
  - "Green Score" gamification to reward sustainable practices.
  
- **For Buyers**:
  - Validated biomass aggregation.
  - Consistent supply chain.

## Tech Stack

- **Frontend**: Next.js 14, React, Tailwind CSS
- **Backend**: Python (FastAPI)
- **AI**: TensorFlow/XGBoost for Price Prediction & Waste Classification

## Getting Started

1. `npm install`
2. `npm run dev`
